# Vibe Coding - Peli ja tekoälyapuri

Tässä vaiheessa oppilaat käyttävät ChatGPT:tä (tai koulun tekoälyä) ja pyytävät tekoälyä muokkaamaan peliä. Näin harjoitellaan tekoälyn hyödyntämistä ohjelmoinnissa.

**Toimi näin:**
1. Avaa pelin koodi (game.js) ja ChatGPT (tai koulun tekoäly).
2. Kopioi jokin alla olevista valmiista promteista ja liitä se tekoälylle.
3. Seuraa tekoälyn ohjeita ja tee muutokset peliin.

## Valmiit promptit

**Promptti A (helppo):**

> Muokkaa koodia niin, että tausta vaihtaa väriä aina kun kerään kolikon.

**Promptti B (helppo+):**

> Lisää peliin ajastin 15 sekuntia. Kun aika loppuu, näytä ‘Peli loppui’ ja lopeta liike.

**Promptti C (hauska):**

> Tee kolikosta ‘pommi’ joka vähentää pisteitä, jos kosken siihen. Pommi ilmestyy välillä satunnaisesti.
